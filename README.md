# MealKit — PWA

## Cómo obtener tu API key
1. Entrá a https://console.anthropic.com/settings/keys
2. Creá una cuenta gratuita si no tenés
3. Hacé click en "Create Key"
4. Copiá la key (empieza con sk-ant-)

## Cómo subir a Vercel (gratis)
1. Creá cuenta en https://vercel.com con tu GitHub
2. Subí esta carpeta a un repo de GitHub
3. En Vercel: "Add New Project" → importar el repo
4. Deploy automático — te da una URL tipo mealkit-xxx.vercel.app

## Cómo instalar en el celular
1. Abrí la URL en Chrome (Android)
2. Menú (⋮) → "Agregar a pantalla de inicio"
3. Confirmar → aparece el ícono como una app normal

## Desarrollo local
```
npm install
npm run dev
```
